1. run docker-compose.yml (database first, then wildfly)
2. add and apply profile in maven settings for wildfly
    <profiles>
        <profile>
            <id>jsf9_flyway</id>
            <properties>
                <flywaydb.url>jdbc:postgresql://localhost:5432/HelloPostgre</flywaydb.url>
                <flywaydb.user>postgres</flywaydb.user>
                <flywaydb.pw>password</flywaydb.pw>
                <flywaydb.schema>public</flywaydb.schema>
            </properties>
        </profile>
    </profiles>
3. build jbpm project
4. build parent
5. deploy
